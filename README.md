# Virtual Garden

Virtual Garden is an interactive web-based gardening game built using HTML, CSS, and JavaScript. Players can plant seeds, water flowers, collect coins, unlock achievements, expand their garden, and discover different flower varieties while weather conditions change.
## Features

### planting system

* Plant flowers using seeds.
* Limited garden space based on owned plots.
* Seed inventory.

### watering system

* Water seedlings to help them grow.
* Plants progress.

### flower collection

* Common flowers
* Rare flowers
* Legendary flowers

Players can discover and collect unique flower types.

### coin system

* Earn coins by growing flowers.
* Different flower rarities provide different rewards:

  * Common Flowers: 10 Coins
  * Rare Flowers: 25 Coins
  * Legendary Flowers: 50 Coins

### weather 

* Sunny
* Rainy
* Rainbow
* Storm

Weather changes automatically and affects the visual appearance of the game.

### butterflies

* Butterflies randomly visit the garden.

### Garden Expansion

* Purchase additional plots using coins.
* Increase garden capacity.

### Flower album

* Automatically records newly discovered flowers.
* Allows players to track their collection progress.

### achievements

Achievements unlock as players reach milestones such as:

* Beginner Gardener
* Flower Lover
* Rich Gardener
* Land Owner
* Seed Collector

### Save system

Game progress is automatically saved using Local Storage:

* Coins
* Seeds
* Garden plots
* Flower collection
* Garden state

Progress remains available after refreshing the page.
